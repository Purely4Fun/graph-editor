1. Скачать архив с проектом
2. Перенести папку react-d3-graph из node_modules подальше от папки с проектом
3. Из корневой папки проекта открыть командную строку
4. Ввести "npm install"
5. Удалить папку react-d3-graph из node_modules и перенести старую копию
6. Для запуска редактора графа ввести в командную строку "npm start"
7. Если появились ошибки при компиляции, ввести в командную строку "npm audit fix"


Для отправки графов на сервер:
1. В редакторе графов нажимаем кнопку "Export plan"
2. Переименовываем файл в "graph.json"
3. Копируем файл в graph-editor-master/src
4. Включаем OpenVPN
5. Запускаем командную строку в в graph-editor-master/src
6. Пишем "node server.js"
7. Заполняем все поля
